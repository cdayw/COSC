## Take all unique letters from 2 strings and output a sorted string to the console
```
def longest(a1, a2):
Alpha = []
for i in a1:
    if i not in Alpha:
        Alpha.append(i)
for i in a2:
    if i not in Alpha:
        Alpha.append(i)
return ''.join(sorted(Alpha))
```

    words = text.split(" ")
    newWords = [word[::-1] for word in words]
    newSentence = " ".join(newWords)
    
    return newSentence
